## ⚠️ Lembretes Permanentes

- ❌ **git commit / git push** — NUNCA execute.
- ❌ **Credenciais hardcoded** — NUNCA exponha.
- 🔐 **Dados PII** — SEMPRE `_hash` + `_encrypted`.
- 📐 **Precisão financeira** — SEMPRE `DECIMAL(12,2)`, nunca `float`.
- 🧪 **Testes** — Nenhuma funcionalidade concluída sem Pest + Playwright.
- 🌐 **Frontend** — NUNCA use `fetch`/`axios` para dados internos; use Inertia.js.
